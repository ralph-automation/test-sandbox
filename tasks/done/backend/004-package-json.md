Create `src/backend/package.json` with:

```json
{
  "name": "ralph-control-center-backend",
  "version": "1.0.0",
  "description": "Backend API for Ralph Control Center",
  "main": "server.js",
  "scripts": {
    "start": "node server.js"
  },
  "dependencies": {
    "express": "^4.18.0",
    "cors": "^2.8.0"
  }
}
```

Create exactly this file. Do not modify any other files.
