Create `src/frontend/components.css` with reusable component styles:

- `.btn` — base button style (padding 8px 16px, border-radius 6px, cursor pointer, border none, font-size 14px)
- `.btn-primary` — accent colored button (var(--accent) bg, white text)
- `.btn-danger` — red button (var(--error) bg, white text)
- `.card` — dark card (var(--bg-secondary) bg, 1px solid var(--border), border-radius 8px, padding 16px)
- `.badge` — small inline badge (padding 2px 8px, border-radius 12px, font-size 12px)
- `.badge-success` — green badge (var(--success) bg)
- `.badge-warning` — yellow badge (var(--warning) bg)
- `.badge-error` — red badge (var(--error) bg)

Use CSS variable names. Do not modify any other files.
