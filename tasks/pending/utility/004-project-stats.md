Create `src/utility/project-stats.sh` — a bash script that prints project statistics:

1. Print "=== Project Stats ==="
2. Count and print number of files in each src/ subdirectory
3. Count total .js files, .html files, .css files, .sh files, .md files
4. Print total lines of code (excluding node_modules/ and .git/)
5. Exit 0

Make the script executable. Do not modify any other files.
